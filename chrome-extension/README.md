# YouTube 批量稍后观看

Chrome 扩展，在 YouTube 频道视频页面批量添加视频到稍后观看列表。

## 功能

- ✅ 在频道视频页面批量选择视频
- ✅ 按时间范围筛选（频道最早 1/2/3/5 年或全部）
- ✅ 跳过已观看的视频
- ✅ 一键添加到稍后观看
- ✅ 无 API 配额限制

## 安装

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `chrome-extension` 文件夹
5. 安装完成

## 使用方法

1. 打开任意 YouTube 频道的视频页面（如 `youtube.com/@channel/videos`）
2. 页面右下角会出现「批量工具」面板
3. 选择加载范围：
   - **全部视频**：加载所有视频
   - **最早 1 年**：只选择频道创建第一年的视频
   - **最早 2 年**：只选择频道前两年的视频
4. 点击「加载视频」等待加载完成
5. 点击「全选」或手动选择视频
6. 点击「添加到稍后观看」

## 时间范围说明

| 选项 | 说明 |
|------|------|
| 全部视频 | 加载并选择所有视频 |
| 最早 1 年 | 频道创建第一年的视频 |
| 最早 2 年 | 频道前两年的视频 |
| 最早 3 年 | 频道前三年的视频 |
| 最早 5 年 | 频道前五年的视频 |
| 按数量 | 加载指定数量的视频后停止 |

筛选逻辑：
- 按年份：先加载所有视频，找到最老视频的日期，然后选择从那天开始 N 年内的视频
- 按数量：加载到指定数量后停止，适合只想处理前 N 个视频的场景

## 注意事项

- 默认跳过已观看的视频（有进度条的），可在设置中关闭
- 添加速度受 YouTube 限制，每批 5 个视频
- 大型频道加载时间较长，请耐心等待

## 问题排查

**工具栏没有出现？**
- 确保在频道视频页面（URL 包含 `/videos`）
- 刷新页面

**页面没有滚动？**
- 按 F12 打开控制台，查看 `[YT批量工具]` 开头的日志
- 尝试手动滚动一下页面再点击加载

**添加失败？**
- 确保已登录 YouTube 账号
- 检查稍后观看列表是否已满（上限约 5000 个）
