# YouTube 批量稍后观看 - 开发日志

## v1.0.0 (当前版本)

### 已实现功能

#### 核心功能
- [x] 在频道视频页面 (`/videos`) 显示批量工具面板
- [x] 自动滚动加载所有视频
- [x] 批量选择视频（复选框）
- [x] 批量添加到稍后观看列表
- [x] 无 API 配额限制（使用 YouTube 内部 API）

#### 时间范围筛选
- [x] 全部视频
- [x] 最早 1 年
- [x] 最早 2 年
- [x] 最早 3 年
- [x] 最早 5 年

筛选逻辑：加载所有视频后，找到最老视频日期，选择从该日期起 N 年内的视频。

#### 其他功能
- [x] 跳过已观看视频（有进度条的）
- [x] 全选/取消全选
- [x] 进度显示
- [x] 暗色主题适配
- [x] SPA 页面切换支持

### 技术实现

#### 视频提取
```javascript
// 从 ytd-rich-item-renderer 元素提取
- videoId: 从链接 href 解析
- title: 从 #video-title 获取
- publishedAt: 从元数据解析相对时间（如 "2年前"）
- isWatched: 检查是否有进度条
- progress: 进度条百分比
```

#### 添加到稍后观看
```javascript
// 使用 YouTube 内部 API
POST https://www.youtube.com/youtubei/v1/browse/edit_playlist
{
  playlistId: 'WL',  // 稍后观看的特殊 ID
  actions: [{ addedVideoId: videoId, action: 'ACTION_ADD_VIDEO' }]
}

// 备用方法：模拟点击菜单
1. 点击视频菜单按钮
2. 点击"保存到播放列表"
3. 勾选"稍后观看"
4. 确认保存
```

#### 已知问题

1. **时间解析不完整**
   - 部分视频的时间文本格式无法解析
   - 日志显示 `解析=null`
   - 原因：YouTube 元数据格式变化或非标准格式

2. **API 请求可能被拦截**
   - `ERR_BLOCKED_BY_CLIENT` 错误
   - 原因：广告拦截器拦截了 youtubei API 请求
   - 解决：备用方法（模拟点击）

3. **添加速度受限**
   - 每批 5 个视频，批次间隔 500ms
   - 大量视频添加较慢
